Binary Search Visualization

Binar qidirish — bu tartiblangan ro'yxatda ma'lum elementni topish uchun ishlatiladigan samarali algoritmdir. U "bo'lib tashla va hukmronlik qil" paradigmasiga asoslanadi.

Eng yaxshi holat: O(1) — agar qidirilayotgan element ro'yxatning o'rtasida bo'lsa. Eng yomon holat: O(log n) — har bir qadamda ro'yxatning yarmidan ko'prog'ini istisno qiladi.

Tezlik: Katta ro'yxatlarda ham samarali ishlaydi, chunki har bir qadamda ro'yxatning yarmidan ko'prog'ini istisno qiladi. Murakkablik: Vaqt murakkabligi O(log n) bo'lib, bu chiziqli qidiruvga nisbatan ancha tez.

Binary search algoritmi kodi 

def binary_search_iterative(arr, target):
 
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = left + (right - left) // 2

        if arr[mid] == target:
            return mid  # Maqsadli element topildi
        elif arr[mid] < target:
            left = mid + 1  # Qidiruvni o'ng yarmida davom ettirish
        else:
            right = mid - 1  # Qidiruvni chap yarmida davom ettirish

    return -1  # Maqsadli element topilmadi

numbers = [2, 5, 8, 12, 16, 23, 38, 56, 72, 91]
target_value = 23
print(binary_search_iterative(numbers, target_value))
