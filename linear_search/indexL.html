Eng yaxshi holat: Agar kerakli element ro'yxatning birinchi elementida bo'lsa, faqat bitta taqqoslash kerak bo'ladi.

Eng yomon holat: Agar kerakli element ro'yxatning oxirida bo'lsa yoki umuman bo'lmasa, barcha elementlarni tekshirish kerak bo'ladi.

Linear search algoritmi kodi 

def linear_search(arr, target):
  
    for index, element in enumerate(arr):
        if element == target:
            return index  # Maqsadli element topildi, uning indeksi qaytariladi
    return -1  # Maqsadli element topilmadi
numbers = [10, 20, 30, 40, 50]
target_value = 30
print(linear_search(numbers, target_value))
