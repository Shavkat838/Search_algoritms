Jump Search Visualization

Jump Search tartiblangan ro'yxatda ma'lum elementni topish uchun ishlatiladigan algoritmidir. U ro'yxatni kichik bloklarga bo'lib, har bir blokning oxirgi elementini tekshirish orqali qidiruvni tezlashtiradi.

Eng yaxshi holat: O(√n) — bloklar bo'yicha qidiruv va blok ichida linear qidiruvning kombinatsiyasi. Eng yomon holat: O(√n) — barcha bloklarni tekshirish va oxirgi blokda linear qidiruvni amalga oshirish.

Tezlik: Katta ro'yxatlar uchun linear qidiruvga nisbatan samaraliroq. Tartibga bog'liqlik: Faoliyatini faqat tartiblangan ro'yxatlarda amalga oshiradi.

Jump search algoritmi kodi 

import math

def jump_search(arr, target):
    n = len(arr)
    step = int(math.sqrt(n))  # Blok o'lchami
    prev = 0

    # Maqsadli elementdan katta yoki unga teng bo'lgan elementni topish uchun bloklarni o'tish
    while prev < n and arr[min(step, n) - 1] < target:
        prev = step
        step += int(math.sqrt(n))
        if prev >= n:
            return -1

    # Maqsadli elementni topish uchun blok ichida chiziqli qidiruv
    for i in range(prev, min(step, n)):
        if arr[i] == target:
            return i

  return -1  # Maqsadli element topilmadi
numbers = [3, 7, 12, 19, 24, 31, 42, 56, 67, 72, 89, 95]
target_value = 42
print(jump_search(numbers, target_value))
